# Factors-Associated-with-Alzheimer-s

## Objective
This project aims to explore and identify factors associated with Alzheimer’s disease using data analysis techniques.

## Dataset
The dataset used is the *Alzheimer’s Disease Dataset* from Kaggle.

It contains various features such as demographic information, lifestyle factors, and health-related indicators that may be associated with Alzheimer’s disease.

This dataset was selected because:
- It includes a variable related to Alzheimer’s condition
- It provides multiple potential influencing factors
- It is structured and suitable for analysis
